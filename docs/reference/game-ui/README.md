# 游戏内 UI 对照截图

这些截图是 GUI 乐器分类、奏法名称和玛勒尼斯音源模式研究的证据，不是运行时资源。

- `instrument-picker-*.png`：游戏“增加乐器”的四个分类页；总数为管乐器 5、弦乐器 11、键盘乐器 6、打击乐器 4。
- `articulation-wind-*.png`、`articulation-strings-*.png`：游戏内音符奏法菜单的对照记录。
- `marnian-instruments-basic-mode.png`：四种玛勒尼斯乐器及其默认 `Basic` 轨道模式。
- `arrangement-drum-kit.png`：架子鼓键位与游戏时间轴的对照。
- `gui-midi-arranger-reference.png`：早期 MIDI 编排界面对照。

玛勒尼斯的 `basic/stereo/super/superoct` 与 synth BNK 的最终波形绑定仍需游戏存档差分与音频 A/B 验证；本目录不把该推断标记为已验证。
