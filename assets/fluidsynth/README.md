Bundled FluidSynth runtime
==========================

The PySide preview uses `pyfluidsynth`, which requires the native FluidSynth
shared library on Windows. This directory contains the official Windows x64
FluidSynth runtime used by the GUI.

Source:
https://github.com/FluidSynth/fluidsynth/releases

Downloaded asset:
`fluidsynth-v2.5.6-win10-x64-cpp11.zip`
