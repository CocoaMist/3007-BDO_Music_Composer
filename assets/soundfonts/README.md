Default preview SoundFont
=========================

`GeneralUser_GS_FluidSynth_v1.44.sf2` is used as the default MIDI preview
SoundFont for the PySide wrapper.

Source:
http://ftp.jaist.jp/pub/osdn.net/sfnet/a/an/androidframe/soundfonts/GeneralUser%20GS%20FluidSynth%20v1.44.sf2

GeneralUser GS license reference:
https://scancode-licensedb.aboutcode.org/generaluser-gs-2.0.html

This file is only used for local preview. BDO export still uses the existing
`tools/midi-to-bdo` conversion code and does not embed audio samples.
